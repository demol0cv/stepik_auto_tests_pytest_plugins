# stepik_auto_tests_pytest_plugins
# How to use
Для установки языка используем аргумент --language
```bash
pytest --language=fr
```
Доступные языки:
```python
langs = [
"ar",
"ca",
"cs",
"da",
"de",
"en-gb",
"el",
"es",
"fi",
"fr",
"it",
"ko",
"nl",
"pl",
"pt",
"pt-br",
"ro",
"ru",
"sk",
"uk",
"zh-hans",
]
```
